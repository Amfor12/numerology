## 🛠 Технический стек и особенности

Проект сверстан с упором на производительность, чистоту кода и современные стандарты веб-разработки:

- **HTML5 / CSS3:** Семантическая разметка и современные стили.
- **Методология БЭМ:** Строгий нейминг классов для легкой поддержки и масштабирования кода в будущем.
- **Flexbox:** Современная и полностью резиновая (адаптивная) сетка, сайт отлично смотрится на любых разрешениях экранов.
- **Vanilla JS (Чистый JavaScript):** Вся логика, интерактивные звезды, табы и кастомные модальные окна написаны на чистом JS без использования тяжелых фреймворков.
- **Оптимизация:** Минимальное количество сторонних зависимостей. Для просмотра изображений подключена только одна легкая библиотека — **Fancybox**.

## 🚀 Как запустить локально

Проект состоит из статических файлов. Для просмотра достаточно скачать репозиторий и открыть файл `index.html` в любом современном браузере. Сборщики (Gulp/Webpack) для запуска не требуются.
